# pyCella

## Description

Just my collection of cellular automata implementations.

* Drossel-Schwabl forest fire model
* Host-Pathogen model
* Game of Life


## Requirements

* python >= 3.11
* uv

## Setup

```shell
uv sync
```

## Run

Drossel-Schwabl forest fire model
```shell
uv run drossel-schwabl
```

Host-Pathogen model
```shell
uv run host-pathogen
```

Game of Life
```shell
uv run game-of-life
```
